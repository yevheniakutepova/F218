Instrukcja uruchomienia bazy danych MongoDB (Docker)

Wymagania:
Zainstalowany Docker
Zainstalowany Node.js i npm
VS Code lub inny edytor

1. Uruchom Docker
Otwórz Docker Desktop i upewnij się, że działa.
2. W katalogu głównym projektu (bibliotekav1) stwórz plik .env i wklej:
MONGO_URI=mongodb://localhost:27017/biblioteka
3. Uruchom MongoDB w Dockerze
Otwórz terminal  i wpisz:
docker run -d --name mongodb -p 27017:27017 -v mongodb_data:/data/db mongo:7.0
MongoDB działa teraz lokalnie na porcie 27017.
Baza danych biblioteka będzie dostępna pod adresem:
mongodb://localhost:27017/biblioteka
4. Zainstaluj paczki Node.js
W terminalu w katalogu projektu wpisz:
npm install express dotenv mongoose
5. Uruchom aplikację
npm start
Otwórz przeglądarkę i wpisz:

http://localhost:3902
6. Zatrzymanie bazy danych

Aby zatrzymać MongoDB w Dockerze, użyj:

docker stop mongodb
docker rm mongodb
